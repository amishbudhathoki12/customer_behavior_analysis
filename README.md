# customer_behavior_analysis
data analytics project showcasing customer behavior analysis using Python, SQL and tableau

# Data Analytics Project

## Overview

This project demonstrates an end-to-end data analytics workflow, starting from data collection and exploration to data cleaning, SQL analysis, visualization, and reporting.

The goal of this project was to extract meaningful insights from the dataset by combining Python-based analysis, PostgreSQL queries, and Tableau dashboard visualization.

---

## Tools & Technologies

- **Python**
  - Pandas

- **Database**
  - PostgreSQL
  - SQL Queries

- **Data Visualization**
  - Tableau

- **Development Environment**
  - VS Code
  - Jupyter Notebook

---

# Project Workflow

## 1. Data Loading

- Imported the dataset into Python using Pandas.
- Checked dataset structure, columns, data types, and initial records.

Example tasks:
- Reading CSV files
- Understanding data shape
- Checking missing values

---

## 2. Exploratory Data Analysis (EDA)

Performed EDA to understand the dataset and discover patterns.

Analysis included:

- Data distribution analysis
- Summary statistics
- Missing value analysis
- Duplicate detection
- Outlier identification
- Relationship between variables

---

## 3. Data Cleaning

Prepared the dataset for analysis by:

- Handling missing values
- Removing duplicate records
- Correcting data types
- Cleaning inconsistent values
- Creating new calculated columns when required

The cleaned dataset was used for further SQL analysis and visualization.

---

## 4. SQL Analysis (PostgreSQL)

Loaded the cleaned dataset into PostgreSQL and performed SQL analysis.

Queries included:

- Data aggregation
- Filtering and sorting
- Grouping data
- Finding trends
- Business-related analysis questions

SQL was used to extract insights efficiently from structured data.

---

## 5. Tableau Dashboard

Created an interactive Tableau dashboard to visualize key findings.

Dashboard features:

- KPI cards
- Charts and graphs
- Filters
- Trend analysis
- Business insights

The dashboard helps users quickly understand important metrics and patterns.

# How to Run This Project

## 1. Clone Repository
git clone <repository-url>
2. Install Required Python Libraries
pip install pandas numpy matplotlib seaborn
3. Run Python Analysis

Open the Jupyter Notebook:

Python/analysis.ipynb

Run all cells to perform data loading, EDA, and cleaning.

4. Setup PostgreSQL
Create a PostgreSQL database
Import the cleaned dataset
Run SQL queries from:
SQL/queries.sql
5. Open Tableau Dashboard

Open:

Tableau/dashboard.twbx

to view the interactive dashboard.
